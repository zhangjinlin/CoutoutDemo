# CoutoutDemo
测试的项目，暂时开放  

这是一个通过flutter调用原生进行抠图的iOS项目。  

以iOS原生项目为根视频的项目，通过cocoapods导入flutter项目。  

flutter是做为原生的一个mudle，通过podinstall 进行集成。  

功能很简单，如需其它功能，请再补充需求！
